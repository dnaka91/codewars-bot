Codewars Bot

Get your Codewars statistics into Slack!

Commands

This are all the available commands the bot understands.

Elements in angle brackets (< and >) are required values used as arguments to the commands.
Elements in brackets ([ and ]) are optional components of the commands and can be omitted.

Add user

Syntax: add <user>

Add a Codewars user to the statistics report.

Remove user

Syntax: remove <user>

Remove a Codewars user from the statistics again.

Stats

Syntax: stats [since <date>]

Show the current statistics of all tracked users.

  • The format of <date> is YYYY/MM/DD, for example 2020/02/12 or 2020/1/2.
  • The date is optional.

Schedule

Syntax: schedule on <weekday> [at <time>]

Set a weekly schedule to send the latest stats.

  • The format of <weekday> is the weekday name in short or long form, for example wed or Friday.
  • The format of <time> is HH:MM, for example 12:15 or 01:00.
  • The time is optional and defaults to 10:00

Notify

Syntax: notify <on|off>

Send notifications whenever new challenges are completed.

Help

Syntax: help

Show this help.
